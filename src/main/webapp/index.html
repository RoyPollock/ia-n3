Aleph Toolbox
Aleph Toolbox (formerly known as IA, by Aleph) is an army list builder tool (and much more) for the miniature game Infinity The Game, by Corvus Belli. The tool has been developed as an open source project, mainly by me (with some collaborations), and it's available for free in both binary and source format. Most images, logos and text/data, used by the tool and in this site, belongs instead to Corvus Belli, used with their permission, and will be removed on their request. Newer logos provided by Vyo of the forums. This site is a clone of Aleph Toolbox with updated profiles for N3 by Roy Pollock. Accuracy is a goal but is not assured.
